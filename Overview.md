1. Ideation_Phase.md
Title: Heart Disease Analysis using Tableau – Ideation Phase
Content:
This project focuses on analyzing heart disease data using Tableau to identify key risk factors such as age, cholesterol, blood pressure, BMI, smoking, and lifestyle habits. The objective is to create interactive dashboards that help doctors, healthcare organizations, and policymakers make informed decisions for early diagnosis and prevention.
